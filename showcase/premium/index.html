NitroMod Features

NitroMod is a Rocket League mod framework with tools for inventory customization, UI personalization, and in-game overlays.

1. Item Spawner

  • Spawn selected items and manual item IDs
  • Bulk spawn support (Spawn X)
  • Quick actions: Spawn Alpha Items, Spawn Hacker Items, Spawn All Item
  • Optional quality settings, random quality, paint ID, series ID, and spawn animation

2. Title Manager

  • Browse and load in-game titles
  • Spawn selected title by ID
  • Spawn all titles in one action
  • Toggle title spawn animation
  • Inventory resync/reset option

3. Rank Changer

  • Change rank by playlist, including division and MMR
  • Supports full rank range up to Supersonic Legend
  • Automatically sets season reward level to max (10) on rank change
  • Reset rank option

4. Rocket Pass Editor

  • Edit Rocket Pass tier
  • Toggle premium ownership flag
  • Edit XP multiplier
  • Apply, cache, and clear values

5. Lobby UI Customization

  • Switch main menu backgrounds (multiple presets)
  • Auto-apply background changes
  • Reset to default background
  • Premium Garage background apply option

6. Fun / Modal Tools

  • Create custom modals (Fun, Ban, Warning)
  • Custom title, body, buttons, and modal colors
  • Ban citation list support
  • Built-in welcome modal

7. In-Game Overlay Modules

  • Car boost circle/text overlay
  • Ball overlay markers
  • Flip reset and jump-ready indicators
  • Additional modules: boost pad timers, ball prediction, boost overlay bar, demo smoke remover, flip detector
